# zkrunyon-ui.github.io

**Zachary K. Runyon — Personal Website**
BYU Information Systems class assignment (IS 201 / personal web portfolio).

Live URL: [https://zkrunyon-ui.github.io/](https://zkrunyon-ui.github.io/)

## File Map

```
/
├── index.html            ← Home — professional landing page
├── about.html            ← Bio & background
├── resume.html           ← Full résumé (semantic HTML, print-ready)
├── contact.html          ← Contact form + map
├── assets/
│   ├── css/
│   │   ├── pro.css       ← Bootstrap overrides for professional pages
│   │   └── scratch.css   ← Hand-rolled stylesheet for scratch.html
│   ├── img/              ← Portrait placeholder + favicons
│   └── js/
│       └── pro.js        ← Shared JS (mobile nav, footer year)
├── scratch/
│   ├── scratch.html      ← From-scratch snowboarding page (no Bootstrap)
│   └── game.html         ← Single-file HTML5 canvas snowboarding game
├── README.md
└── .nojekyll             ← Prevents GitHub Pages from running Jekyll
```
